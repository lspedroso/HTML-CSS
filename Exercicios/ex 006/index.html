Principais formatações

Negrito

Para deixar um texto em negrito, basta usar a tag <strong> ou a tag <b>.

Exemplo: Olá, mundo! ou Olá, mundo!

Itálico

Para deixar um texto em itálico, basta usar a tag <em> ou a tag <i>.

Exemplo: Olá, mundo! ou Olá, mundo!

Sublinhado

Para deixar um texto sublinhado, basta usar a tag <u>.

Exemplo: Olá, mundo!

Tachado

Para deixar um texto tachado, basta usar a tag <s>.

Exemplo: Olá, mundo!

Destacado

Para deixar um texto destacado, basta usar a tag <mark>.

Exemplo: Olá, mundo!
